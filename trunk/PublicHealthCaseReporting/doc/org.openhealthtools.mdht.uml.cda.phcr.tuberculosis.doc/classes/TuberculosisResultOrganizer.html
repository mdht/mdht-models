Tuberculosis Result Organizer

[Organizer: templateId 2.16.840.1.113883.10.20.15.3.21]

  1. SHALL conform to PHCR Result Organizer template (templateId: 2.16.840.1.113883.10.20.15.3.59)
  2. SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet 2.16.840.1.114222.4.11.3205 Lab Test Result Name (TB) STATIC 1 (CONF-397, CONF:1999, CONF:952)
  3. SHALL contain at least one [1..*] component (CONF:1231), such that
    1. Contains exactly one [1..1] Tuberculosis Result Observation (templateId: 2.16.840.1.113883.10.20.15.3.13)

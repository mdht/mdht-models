Tuberculosis Case Report

[ClinicalDocument: templateId 2.16.840.1.113883.10.20.15.1.1]

  1. SHALL conform to PHCR Public Health Case Report template (templateId: 2.16.840.1.113883.10.20.15)
  2. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] Tuberculosis PHCR Clinical Information Section (templateId: 2.16.840.1.113883.10.20.15.2.5)
  3. SHOULD contain zero or one [0..1] component, such that
    1. Contains exactly one [1..1] Tuberculosis PHCR Relevant Dx Tests Section (templateId: 2.16.840.1.113883.10.20.15.2.6)
  4. SHOULD contain zero or one [0..1] component, such that
    1. Contains exactly one [1..1] Tuberculosis PHCR Social History Section (templateId: 2.16.840.1.113883.10.20.15.2.7)
  5. SHOULD contain zero or one [0..1] component, such that
    1. Contains exactly one [1..1] Tuberculosis PHCR Treatment Information Section (templateId: 2.16.840.1.113883.10.20.15.2.8)
  6. MAY contain zero or one [0..1] component, such that
    1. Contains exactly one [1..1] Tuberculosis PHCR Immunization Section (templateId: 2.16.840.1.113883.10.20.15.2.23)
  7. SHALL contain exactly one [1..1] title = "Public Health Case Report - Tuberculosis" (CONF:569)

Response Section

[Section: templateId 2.16.840.1.113883.17.3.10.1.3]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.17.3.10.1.3"
  2. SHALL contain exactly one [1..1] code (CONF:10573)/@code="67664-3" EMS Response Section (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10572)
  3. SHALL contain exactly one [1..1] title and SHOULD equal "EMS Response" (CONF:10575)
  4. SHALL contain exactly one [1..1] text (CONF:10574)
  5. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Delay Organizer (templateId: 2.16.840.1.1133883.17.3.10.1.95)
  6. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Dispatch Location Organizer (templateId: 2.16.840.1.1133883.17.3.10.1.96)
  7. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Response Odometer Reading Organizer (templateId: 2.16.840.1.1133883.17.3.10.1.97)
  8. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Response Mode To Scene (templateId: 2.16.840.1.1133883.17.3.10.1.98)

[Section: templateId 2.16.840.1.113883.17.3.10.1.9]

      1. Such observations SHALL contain exactly one [1..1] code (CONF:10768)/@code="10154-3" EMS complaint (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10769)

        NEMSIS Trace: ESituation.04

      3. Such observations MAY contain zero or one [0..1] entryRelationship (CONF:10771)
        1. This entryRelationship Contains exactly one [1..1] observation

            NEMSIS Trace: ESituation.05, 06
            Units SHALL be selected from UCUM time units (i.e., s, min, h, d, wk, mo, a)

      4. Such observations MAY contain zero or one [0..1] entryRelationship (CONF:10772)
        1. This entryRelationship Contains at least one [1..*] observation

            NEMSIS Trace: ESituation.03

      5. Such observations SHOULD contain exactly one [1..1] targetSiteCode (CONF:10773), where the @code SHALL be selected from ValueSet AnatomicLocation 2.16.840.1.113883.17.3.5.76 STATIC (CONF:10774)

        NEMSIS Trace: ESituation.07
        Complaint site is required if complaint is type "Chief complaint"

      6. Such observations SHOULD contain zero or one [0..1] entryRelationship (CONF:10775)

        The organ system is required if the complaint is of type "Chief complaint"

        1. This entryRelationship SHALL contain exactly one [1..1] observation (CONF:10782)
          1. This observation SHALL contain exactly one [1..1] code (CONF:10784)/@code="69468-7" Organ system (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10783)

            NEMSIS Trace: ESituation.08

        The organ system is required if the complaint is of type "Chief complaint"

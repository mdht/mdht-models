Glasgow Coma Score Organizer

[Organizer: templateId 2.16.840.1.1133883.17.3.10.1.170]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.1133883.17.3.10.1.170"
  2. SHALL contain exactly one [1..1] component (CONF:10313)
    1. Contains exactly one [1..1] Glasgow Eye (templateId: 2.16.840.1.1133883.17.3.10.1.160)
  3. SHALL contain exactly one [1..1] component (CONF:10314)
    1. Contains exactly one [1..1] Glasgow Motor (templateId: 2.16.840.1.1133883.17.3.10.1.176)
  4. MAY contain zero or one [0..1] component (CONF:10315)
    1. Contains exactly one [1..1] Glasgow Qualifier (templateId: 2.16.840.1.1133883.17.3.10.1.159)
  5. SHOULD contain zero or one [0..1] component (CONF:10316)
    1. Contains exactly one [1..1] Glasgow Total (templateId: 2.16.840.1.1133883.17.3.10.1.158)
  6. SHALL contain exactly one [1..1] component (CONF:10317)
    1. Contains exactly one [1..1] Glasgow Verbal (templateId: 2.16.840.1.1133883.17.3.10.1.162)
  7. MAY contain zero or one [0..1] effectiveTime

    • NEMSIS Trace: EVitals.01

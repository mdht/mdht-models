Complaint Reported By Dispatch

[Observation: templateId 2.16.840.1.1133883.17.3.10.1.74]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.1133883.17.3.10.1.74"
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code (CONF:10568)/@code="67570-2" Complaint reported by dispatch (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10569)
  4. SHALL contain exactly one [1..1] value with data type CD (CONF:10571), where the @code SHALL be selected from ValueSet Complaint 2.16.840.1.113883.17.3.5.49 STATIC (CONF:10570)

    • NEMSIS Trace: EDispatch.01

EMS Current Medication Section

[Section: templateId 2.16.840.1.113883.17.3.10.1.15]

NEMSIS Trace: EHistory.12

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.17.3.10.1.15"
  2. SHALL contain exactly one [1..1] code/@code="67844-1" EMS Current Medications Section (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10218)
  3. SHALL contain exactly one [1..1] title and SHOULD equal "EMS Current Medications Section" (CONF:10219)
  4. SHALL contain exactly one [1..1] text
  5. SHALL contain exactly one [1..1] entry (CONF:10220)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain zero or one [0..1] observation (CONF:10223)
      1. This observation SHALL contain exactly one [1..1] code/@code="67791-4" Currently on medication (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10224)
      2. This observation SHALL contain exactly one [1..1] value with data type BL (CONF:10225)

        If EHistory.12 has the value "None Reported," the value is "False"
        If EHistory.12 has the value "Refused" or "Unable to Complete," the value is "NI"; the
        original value may be recorded in Observation.text.

  6. SHALL contain exactly one [1..1] entry (CONF:10221)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain exactly one [1..1] observation (CONF:10226)
      1. This observation SHALL contain exactly one [1..1] code/@code="69749-0" Patient on anticoagulants (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10227)
      2. This observation SHALL contain exactly one [1..1] value with data type BL (CONF:10228)

        NEMSIS Trace: If EHistory.12 has the value "No anticoagulants," the value is False

  7. SHALL contain exactly one [1..1] entry (CONF:10222)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry MAY contain zero or more [0..*] substanceAdministration (CONF:10229)
      1. Such substanceAdministrations MAY contain zero or one [0..1] @negationInd (CONF:10233)

        If EHistory.12 has the value "None Reported," the value is "True"
        This negation of a notional class is parallel to the explicit question "patient on medication"

      2. Such substanceAdministrations MAY contain zero or one [0..1] @nullFlavor (CONF:10234)

        If EHistory.12 has the value "Refused" or "Unable to Complete," the value is "NI"; the
        original value may be recorded in SubstanceAdministration.text.
        This nulled notional class is parallel to the explicit question "patient on medication"

      3. Such substanceAdministrations SHOULD contain zero or one [0..1] doseQuantity (CONF:10230)

        NEMSIS Trace: EHistory.13, EHistory.14

      4. Such substanceAdministrations SHOULD contain zero or one [0..1] routeCode (CONF:10232), where the @code SHALL be selected from ValueSet MedicationAdministrationRoute 2.16.840.1.113883.17.3.5.43 STATIC (CONF:10231)

        NEMSIS Trace: EMedication.15

      5. Such substanceAdministrations SHALL contain exactly one [1..1] consumable (CONF:10235)
        1. This consumable SHALL contain exactly one [1..1] manufacturedProduct (CONF:10236)
          1. This manufacturedProduct SHALL contain exactly one [1..1] manufacturedLabeledDrug (CONF:10237)
            1. This manufacturedLabeledDrug SHALL contain exactly one [1..1] code (CONF:10239), where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.6.88 RxNorm) (CONF:10238)

              NEMSIS Trace: EHistory.12

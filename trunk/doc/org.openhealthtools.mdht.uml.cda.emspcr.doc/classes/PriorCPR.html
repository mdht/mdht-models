Prior CPR

[Observation: templateId 2.16.840.1.1133883.17.3.10.1.125]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.1133883.17.3.10.1.125"
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="67506-6" CPR prior to EMS arrival (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10163)
  4. SHALL contain exactly one [1..1] value with data type BL (CONF:10164)

    • NEMSIS Trace: EArrest.05

  5. MAY contain zero or one [0..1] participant (CONF:10165)
    1. This participant SHALL contain exactly one [1..1] @typeCode="RESP"
    2. This participant SHALL contain zero or one [0..1] participantRole
      1. This participantRole SHALL contain exactly one [1..1] code, where the @code SHALL be selected from ValueSet PriorCareProviderRole 2.16.840.1.113883.17.3.5.12 STATIC (CONF:10167)

        NEMSIS Trace: eArrest.06

  </participant>
</observation>

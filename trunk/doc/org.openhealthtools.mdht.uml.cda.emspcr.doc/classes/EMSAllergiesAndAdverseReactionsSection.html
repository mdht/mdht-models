EMS Allergies And Adverse Reactions Section

[Section: templateId 2.16.840.1.113883.17.3.10.1.13]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.17.3.10.1.13"
  2. SHALL contain exactly one [1..1] code/@code="67841-7" EMS Allergies and Adverse Reactions Section (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10196)
  3. SHALL contain exactly one [1..1] title and SHOULD equal "EMS Allergies and Adverse Reactions"
  4. SHALL contain exactly one [1..1] text
  5. SHALL contain exactly one [1..1] entry (CONF:10198)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain zero or one [0..1] observation
      1. This observation SHALL contain exactly one [1..1] code/@code="67794-8" Existence of drug allergy (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10203)
      2. This observation SHALL contain exactly one [1..1] value with data type BL (CONF:10204)

        If NEMSIS EHistory.06 is "No Known Drug Allergies," the value should be FALSE. If it is a proper value, the value should be TRUE. If it is a flavor of null, the appropriate flavor should be used here. See the null table to translate NEMSIS flavors into HL7; original values may be stored in Observation.text.

    3. This entry MAY contain zero or one [0..1] observation
      1. This observation SHALL contain exactly one [1..1] code (CONF:10205)/@code="67850-8" Drug allergy (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10206)
      2. This observation SHALL contain exactly one [1..1] value with data type CD/@code="106190000" allergy (CodeSystem: 2.16.840.1.113883.6.96 SNOMEDCT) (CONF:10207)
      3. This observation SHALL contain exactly one [1..1] participant
        1. This participant SHALL contain exactly one [1..1] participantRole (CONF:10208)
          1. This participantRole SHALL contain exactly one [1..1] playingEntity (CONF:10209)
            1. This playingEntity SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.6.88 RxNorm) (CONF:10210)

              NEMSIS Trace: EHistory.06

  6. SHALL contain exactly one [1..1] entry (CONF:10199)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain exactly one [1..1] organizer (CONF:10211)
      1. This organizer SHALL contain exactly one [1..1] component (CONF:10212)
        1. This component SHALL contain exactly one [1..1] code/@code="69747-4" EMS Existence of environmental allergy (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10214)
        2. This component SHALL contain exactly one [1..1] value with data type BL (CONF:10215)

          If NEMSIS EHistory.07 has no values, this is FALSE; otherwise it is TRUE.

      2. This organizer MAY contain zero or more [0..*] component (CONF:10213)
        1. Such components SHALL contain exactly one [1..1] code/@code="69748-2" Known allergies to food or environmental agents (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10216)

          Required if Existence is True

        2. Such components SHALL contain exactly one [1..1] value with data type CD, where the @code SHALL be selected from ValueSet EnvironmentalAllergyType 2.16.840.1.113883.17.3.5.83 STATIC (CONF:10217)

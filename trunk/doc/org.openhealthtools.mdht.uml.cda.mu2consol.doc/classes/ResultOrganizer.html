Result Organizer

[Organizer: templateId 2.16.840.1.113883.10.20.22.4.1]

Known Subclasses: Cognitive Status Result Organizer, Functional Status Result Organizer

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.1"
  2. SHALL conform to Consol Result Organizer template (templateId: 2.16.840.1.113883.10.20.22.4.1)
  3. SHALL contain at least one [1..*] component (CONF:7124, CONF:14850)
    1. Contains exactly one [1..1] Result Observation (templateId: 2.16.840.1.113883.10.20.22.4.2)

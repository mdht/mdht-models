Body System or Organ Class

[Observation: templateId 2.16.840.1.113883.10.20.23.54]

The resuable structure provides for entry of the body system or organ class that is relevant for a measurement or event. This structure is captured as a template to facilitate its reuse as a characteristic of different types of data within clinical trial reporting.

  1. SHALL contain exactly one [1..1] @classCode
  2. SHALL contain exactly one [1..1] @moodCode
  3. SHALL contain exactly one [1..1] code
  4. SHALL contain zero or one [0..1] value with data type CE
    • Provide the Body System or Organ class code. [SDTM: various]

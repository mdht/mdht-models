Start Relative to Reference Period

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

A resusable structure that captures the relationship between the start of an activity and the reference period defined within the demographics domain.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • The fixed code value indicates the observation captures information regarding the start of an activity relative to the previously defined reference period.
  4. SHALL contain exactly one [1..1] value with data type CD, where the @code SHALL be selected from (CodeSystem: Reference Period Relationship)
    • Information on the timing of an activity's start with respect to the previously defined reference period.

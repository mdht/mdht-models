Event Pattern

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

Information that indicates the pattern of an event over time.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="C83208" Adverse Event Pattern (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • A code that indicates the observation contains information regarding the pattern of the adverse event over time.
  4. SHALL contain exactly one [1..1] value with data type CE
    • Provides information on the pattern of the event over time. [SDTM: AEPATT], [Study Data Tabulation Model: -PATT]

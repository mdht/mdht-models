Dose Adjustment Reason

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

A description of the reason for, or explanation of, the adjustment of the administered dosage.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. Contains exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
  4. SHALL contain exactly one [1..1] value with data type ED
    • A text entry describing the reason for a dose adjustment.

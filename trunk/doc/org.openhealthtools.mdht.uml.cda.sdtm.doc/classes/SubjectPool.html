Subject Pool

[Observation: templateId 2.16.840.1.113883.10.20.23.53]

Definition of a pool (group) that is created in order to allow findings that refer to a group of subjects rather than to a single one.

  1. Contains exactly one [1..1] @classCode
  2. Contains exactly one [1..1] @moodCode
  3. Contains exactly one [1..1] code
  4. Contains zero or more [0..*] id
  5. Contains zero or more [0..*] value

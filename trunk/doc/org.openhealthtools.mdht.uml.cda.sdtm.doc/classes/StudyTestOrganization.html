Study Test Organization

[Participant2: templateId 2.16.840.1.113883.10.20.23.73]

Information on the organization performing a test or other activity within the context of the study. Within SDTM, this is referred to as a "vendor".

  1. SHALL contain exactly one [1..1] @typeCode="AUT"
  2. SHALL contain exactly one [1..1] participantRole
    1. This participantRole SHALL contain zero or one [0..1] @classCode="ASSIGNED"
    2. This participantRole SHALL contain zero or one [0..1] performer
      1. This performer SHALL contain zero or one [0..1] @classCode="ORG"
      2. This performer SHALL contain zero or one [0..1] @determinerCode="INSTANCE"
      3. This performer SHALL contain exactly one [1..1] name

        The name of the organization ("vendor") that performed the test. [SDTM: LBNAM, EGNAM, MBNAM, MSNAM, PCNAM], [SEND: LBNAM, MANAM, MINAM, PCNAM, EGNAM, TSTFNAM], [Study Data Tabulation Model: -NAM]

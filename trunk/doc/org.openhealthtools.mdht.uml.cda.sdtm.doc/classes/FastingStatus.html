Fasting Status

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

Information regarding whether or not the subject was fasting at the effective time - when the specimen was drawn - of the test producing the finding.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value indicates the observation concerns the subject's fasting status.
  4. SHALL contain exactly one [1..1] value with data type BN
    • Information on whether the person was fasting at the effective time for the test. [SDTM: LBFAST, PCFAST], [SEND: LBFAST, PCFAST, BWFAST], [Study Data Tabulation Model: FAST]

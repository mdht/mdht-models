Progress Note Document Type Code

Value Set ProgressNoteDocumentTypeCode - 2.16.840.1.113883.11.20.8.1
Code System LOINC - 2.16.840.1.113883.6.1
Source HL7
Definition This value set includes the preferred code and pre-coordinated LOINC codes that have the scale DOC (document) and a 'component' referring to "subsequent evaluation notes". Although these pre-coordinated LOINC codes are available for use, we recommend the preferred code (11506-3 Progress Note). When these pre-coordinated codes are used, any coded values describing the author or performer of the service act or the practice setting must be consistent with the LOINC document type. Note: "Subsequent evaluation note" is equivalent to Progress Note.

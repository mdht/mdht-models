Personal Relationship Role Type

Value Set Personal Relationship Role Type - 2.16.840.1.113883.1.11.19563
Code System RoleCode - 2.16.840.1.113883.5.111
Version 1
Definition A Personal Relationship records the role of a person in relation to another person. This value set is to be used when recording the relationships between different people who are not necessarily related by family ties, but also includes family relationships

Social History

[Observation: templateId 2.16.840.1.113883.10.20.1.33]

  1. SHALL conform to IHE Social History Observation template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.13.4)
  2. SHOULD contain zero or more [0..*] code, which SHOULD be selected from ValueSet 2.16.840.1.113883.3.88.12.80.60 Social History Type STATIC 1
  3. SHOULD contain zero or one [0..1] effectiveTime
  4. SHALL contain exactly one [1..1] text

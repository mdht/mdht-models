Birth Attendant Titles (NCHS)

Value Set Birth Attendant Titles (NCHS) - 2.16.840.1.114222.4.11.7111
Version Dynamic
Source PHIN VADS
Source URL https://phinvads.cdc.gov/vads/ViewValueSet.action?oid=2.16.840.1.114222.4.11.7111
Definition The value set contains the list of titles used by persons (attendant) responsible for delivering a child/fetus. The attendant at birth/delivery is defined as the individual physically present at the delivery who is responsible for the delivery.

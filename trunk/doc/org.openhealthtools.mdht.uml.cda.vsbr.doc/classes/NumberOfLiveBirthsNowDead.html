Number of Live Births now Dead

[Observation: templateId 2.16.840.1.113883.10.20.26.38]

The total number of previous live-born infants now dead.  For multiple deliveries include all live-born infants before this infant in the pregnancy who are now dead.  If the first born, do not include this infant.

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.26.38"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:215)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:217)
  4. SHALL contain exactly one [1..1] code (CONF:216)/@code="68496-9" Live births now dead (CodeSystem: 2.16.840.1.113883.6.1 LOINC)

    • A code value that indicates the nature of the observation - it records the total number of previous live-born infants now dead.

  5. SHALL contain exactly one [1..1] value with data type INT (CONF:218)

    • The total number of previous live-born infants now dead.  The entry is a non-negative integer within the range from zero to 30.

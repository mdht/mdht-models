Number of Live Births now Dead

[Observation: templateId null]

The total number of previous live-born infants now dead. For multiple deliveries include all live-born infants before this infant in the pregnancy who are now dead. If the first born, do not include this infant.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • A code value that indicates the observation contains the total number of previous live-born infants now dead.
  4. SHALL contain zero or more [0..*] value with data type INT
    • The total number of previous live-born infants now dead.

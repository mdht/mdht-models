Attempted Vacuum Extraction

[Observation: templateId null]

Information on whether an unsucessful attempt at a vaccum extraction was made.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • Indicates the observation relates to an attempted vacuum extraction.
  4. SHALL contain exactly one [1..1] value with data type BL
    • Was delivery with vacuum extraction attempted but unsuccessful?

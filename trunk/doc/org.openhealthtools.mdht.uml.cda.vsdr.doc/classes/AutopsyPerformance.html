Autopsy Performance

[Observation: templateId 2.16.840.1.113883.10.20.26.1.2]

Information on the performance of an autopsy.  Minimally, information on whether or not an autopsy was performed is recorded.

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.26.1.2"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  4. SHALL contain exactly one [1..1] code/@code="21986-5" Autopsy Status (CodeSystem: 2.16.840.1.113883.6.1 LOINC)

    • The code value notes that autopsy status information is provided.

  5. SHALL contain exactly one [1..1] effectiveTime

    • If an autopsy is performed, this field is valued with the date and time the autopsy was begun and completed.  Start date/time for the autopsy is recorded using the low property of the interval data type, while the completion date and time are recorded using the high property of the data type.  Note, normally only the completion date and time will be provided.

  6. SHALL contain exactly one [1..1] value with data type BL

    • An indicator that tells whether an autopsy was performed.

  7. SHALL contain zero or one [0..1] performer
    1. This performer SHALL contain zero or one [0..1] @typeCode="PRF"
    2. This performer SHALL contain zero or one [0..1] assignedEntity
      1. This assignedEntity SHALL contain zero or one [0..1] @classCode="ASSIGNED"
      2. This assignedEntity SHALL contain zero or one [0..1] assignedPerson
        1. This assignedPerson SHALL contain zero or one [0..1] @classCode="PSN"
        2. This assignedPerson SHALL contain zero or one [0..1] @determinerCode="INSTANCE"
        3. This assignedPerson SHALL contain exactly one [1..1] name

          This field is valued with the name of the person who performed the autopsy.

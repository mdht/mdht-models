Death Report Document Body

[Section: templateId 2.16.840.1.113883.10.20.24.1.2]

  1. SHALL contain zero or one [0..1] code (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    • A code to indicate this is the section containing death reporting information.
  2. Contains zero or one [0..1] text
    • Contains, as text entries that can be rendered using a browser, the content of the structured information from the entries within the section.
  3. SHALL contain exactly one [1..1] time of Death
    1. Contains exactly one [1..1] Time of Death (templateId: 2.16.840.1.113883.10.20.24.1.3)
  4. SHALL contain exactly one [1..1] location of Death
    • The report is expected to capture information about the physical location of death, either as an address or as a description.
    1. Contains exactly one [1..1] Location of Death (templateId: 2.16.840.1.113883.10.20.24.1.4)
  5. MAY contain zero or one [0..1] death Certification
    1. Contains exactly one [1..1] Death Certification (templateId: 2.16.840.1.113883.10.20.24.1.5)
  6. SHALL contain exactly one [1..1] manner of Death
    1. Contains exactly one [1..1] Manner of Death (templateId: 2.16.840.1.113883.10.20.24.1.7)
  7. MAY contain zero or one [0..1] pregnancy Status
    1. Contains exactly one [1..1] Pregnancy Status (templateId: 2.16.840.1.113883.10.20.24.1.8)
  8. SHOULD contain zero or one [0..1] tobacco Use
    1. Contains exactly one [1..1] Tobacco Use (templateId: 2.16.840.1.113883.10.20.24.1.9)
  9. SHOULD contain zero or one [0..1] injury
    1. Contains exactly one [1..1] Injury (templateId: 2.16.840.1.113883.10.20.24.1.10)
  10. SHALL contain exactly one [1..1] cause of Death
    1. Contains exactly one [1..1] Death Causal Information (templateId: 2.16.840.1.113883.10.20.24.1.6)
  11. SHALL contain exactly one [1..1] autopsy Performance
    1. Contains exactly one [1..1] Autopsy Performance (templateId: 2.16.840.1.113883.10.20.24.1.11)
  12. MAY contain zero or one [0..1] autopsy Results
    1. Contains exactly one [1..1] Autopsy Results (templateId: 2.16.840.1.113883.10.20.24.1.13)
  13. MAY contain zero or one [0..1] coroner Referral
    1. Contains exactly one [1..1] Coroner Referral (templateId: 2.16.840.1.113883.10.20.24.1.14)
  14. SHALL contain exactly one [1..1] coroner Case Transfer
    1. Contains exactly one [1..1] Coroner Case Transfer (templateId: 2.16.840.1.113883.10.20.24.1.12)
  15. SHALL contain exactly one [1..1] deathLocationType

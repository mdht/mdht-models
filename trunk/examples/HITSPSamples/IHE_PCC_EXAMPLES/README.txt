Title: IHE PCC Examples Package

Description: This package contains the example XML documents created against the guidelines specified by the IHE Patient Care Coordination (PCC) Technical Framework.  The included sample documents depict fictional health summaries. Any resemblance to a real person or a real medical summary is coincidental.  These documents are intended to demonstrate syntactical conformance only.  No effort has been made to create coherent or meaningful clinical or demographic information.

Release Date: February 10, 2009


Package Contents:

sampleApe.xml -- XML example file for IHE PCC Antepartum Education (1.3.6.1.4.1.19376.1.5.3.1.1.16.1.3)

sampleAphp.xml -- XML example file for IHE PCC Antepartum History and Physical (1.3.6.1.4.1.19376.1.5.3.1.1.16.1.1)

sampleAps.xml  -- XML example file for IHE PCC Antepartum Summary (1.3.6.1.4.1.19376.1.5.3.1.1.11.2)

sampleEdComposite.xml -- XML example file for IHE PCC Emergency Department Composite Triage and Nursing Note (1.3.6.1.4.1.19376.1.5.3.1.1.13.1.3)

sampleNursingNote.xml -- XML example file for IHE PCC Emergency Department Nursing Note (1.3.6.1.4.1.19376.1.5.3.1.1.13.1.2)

sampleEdPhysicianNote.xml -- XML example file for IHE PCC Emergency Department Physician Note (1.3.6.1.4.1.19376.1.5.3.1.1.13.1.4)

sampleEdReferral.xml -- XML example file for IHE PCC Emergency Department Referral Note (1.3.6.1.4.1.19376.1.5.3.1.1.10)

sampleTriageNote.xml -- XML example file for IHE PCC Emergency Department Triage Note (1.3.6.1.4.1.19376.1.5.3.1.1.13.1.1)

sampleImmunizationDetail.xml -- XML example file for IHE PCC Immunization Detail (1.3.6.1.4.1.19376.1.5.3.1.1.18.1.2)

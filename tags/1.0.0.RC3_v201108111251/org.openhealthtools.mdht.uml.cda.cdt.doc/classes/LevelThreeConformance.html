Level Three Conformance

[ClinicalDocument: templateId 2.16.840.1.113883.10.20.30]

  1. SHALL conform to Level Two Conformance template (templateId: 2.16.840.1.113883.10.20.20)
  1. SHALL conform to Level One Conformance template (templateId: 2.16.840.1.113883.10.20.10)
  2. SHALL conform to Level Two Conformance template (templateId: 2.16.840.1.113883.10.20.20)
  3. Contains exactly one [1..1] typeId, where its data type is InfrastructureRootTypeId
  4. Contains exactly one [1..1] id, where its data type is II
  5. Contains exactly one [1..1] code, where its data type is CE
  6. Contains exactly one [1..1] effectiveTime, where its data type is TS
  7. Contains exactly one [1..1] confidentialityCode, where its data type is CE
  8. Contains at least one [1..*] recordTarget, where its type is Record Target
  9. Contains at least one [1..*] author, where its type is Author
  10. Contains exactly one [1..1] custodian, where its type is Custodian
  11. Contains exactly one [1..1] component, where its type is Component2

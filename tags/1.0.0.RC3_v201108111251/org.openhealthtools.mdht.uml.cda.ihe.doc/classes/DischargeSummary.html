Discharge Summary

[ClinicalDocument: templateId 1.3.6.1.4.1.19376.1.5.3.1.1.4]

  1. SHALL conform to Medical Summary template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.1.2)
  2. SHALL contain exactly one [1..1] code (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  3. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] Active Problems Section (templateId: 1.3.6.1.4.1.19376.1.5.3.1.3.6)

Encounters Activity

[Encounter: templateId 2.16.840.1.113883.10.20.1.21]

  1. SHALL contain exactly one [1..1] @classCode="ENC" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain at least one [1..*] id
  4. SHOULD contain exactly one [1..1] code, which SHOULD be selected from ValueSet 2.16.840.1.113883.1.11.13955 EncounterCode STATIC
  5. MAY contain zero or one [0..1] effectiveTime

Immunization

[SubstanceAdministration: templateId 2.16.840.1.113883.3.88.11.83.13]

  1. SHALL conform to IHE Immunization template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.12)
  2. SHALL contain zero or one [0..1] code (CodeSystem: 2.16.840.1.113883.12.292 Vaccines administered (CVX))
  3. SHOULD contain zero or one [0..1] entryRelationship, such that
  4. The reason for refusal SHALL be coded as specified in HITSP/C80 Section 2.2.3.5.3 No Immunization Reason

Medications Section

[Section: templateId 2.16.840.1.113883.3.88.11.83.112]

The Medications Section contains information about the relevant medications for the patient. At a minimum, the currently active medications should be listed.

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.3.88.11.83.112"
  2. SHALL conform to IHE Medications Section template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.3.19) (C83-[CT-112-1])
  3. SHALL contain at least one [1..*] entry (C83-[CT-112-2])
    1. Contains exactly one [1..1] Medication (templateId: 2.16.840.1.113883.3.88.11.83.8)

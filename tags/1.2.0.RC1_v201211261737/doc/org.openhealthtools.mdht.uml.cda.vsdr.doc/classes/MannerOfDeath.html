Manner of Death

[Observation: templateId 2.16.840.1.113883.10.20.24.1.7]

This template contains information on the main categories in which death can occur.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="LOINC TBD" Manner of Death (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  4. SHALL contain exactly one [1..1] value (CodeSystem: OID TBD Manners Of Death), where its data type is CE
    • A coded indication of the manner in which the person died.

Location of Death

[Observation: templateId 2.16.840.1.113883.10.20.24.1.4]

This template contains information to document where the death occurred.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="LOINC TBD" Location of Death (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  4. MAY contain zero or one [0..1] text
    • Information about the place where death occurred. It is provided if no address can be.
  5. SHALL contain exactly one [1..1] value, where its data type is AD
    • The mailing address for the place where the person died. This attribute is collected if the person died at a home, a health facility, or other location with a postal address.

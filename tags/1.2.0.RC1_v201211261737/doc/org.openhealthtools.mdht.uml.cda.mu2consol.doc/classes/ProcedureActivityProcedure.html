Procedure Activity Procedure

[Procedure: templateId 2.16.840.1.113883.10.20.22.4.14]

  1. SHALL conform to Consol Procedure Activity Procedure template (templateId: 2.16.840.1.113883.10.20.22.4.14)
  2. SHALL contain exactly one [1..1] code (CONF:7656)
  3. 'code' SHOULD be selected from SNOMED CT (codeSystem 2.16.840.1.113883.6.96) or CPT-4 (CodeSystem 2.16.840.1.113883.6.12)

Code System NDF-RT (Drug Classification) - 2.16.840.1.113883.3.26.1.5
Description This identifies the pharmacological drug class, such as Cephalosporins. Shall contain a value descending from the NDF-RT concept types of "Mechanism of Action - N0000000223", "Physiologic Effect - N0000009802" or "Chemical Structure - N0000000002".

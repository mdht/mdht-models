Related Record

[Act: templateId 2.16.840.1.113883.10.20.23.xxxx]

Use this structure to define the relationship between an information item and another item or group of items.

  1. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • An indication of the nature of the relationship, specifically of its cardinality. [SDTM: RELTYPE], [SEND: RELTYPE]
  4. SHALL contain at least one [1..*] id
    • An identifier of the record that is being related. The root property of the II type is used to indicate the namespace for the identifier, in particular whether it is a group id or a sequence number. [SDTM: RELID], [SEND: RELID]

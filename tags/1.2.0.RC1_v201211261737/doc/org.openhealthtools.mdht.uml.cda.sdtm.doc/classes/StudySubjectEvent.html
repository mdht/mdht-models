Study Subject Event

[Observation: templateId 2.16.840.1.113883.10.20.23.102]

The structure captures planned protocol milestones such as randomization and study completion, and occurrences, conditions, or incidents independent of planned study evaluations occurring during the trial (e.g., adverse events) or prior to the trial (e.g., medical history). It is included to make allowances for the creation of event based domains that are not currently modeled within the SEND or SDTM implementation guides.

  1. Contains exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="C49562" Adverse Event - Reported (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value indicates the nature of the observation. The verbatim term is captured as original text. A modified term is representated as a translation. [Study Data Tabulation Model: -TERM, -MODIFY, -DCOD]
  4. SHOULD contain zero or one [0..1] effectiveTime
    • Used to capture the start date/time of the event, its end date/time, and/or its duration. [SDTM: AESTDTC, AEENDTC, AEDUR]
  5. SHALL contain exactly one [1..1] value with data type CE
    • The nature of the event.
    1. SHALL satisfy: Capture the dictionary derived term for the adverse event using the code and codeSystem properties of the CE data type.
  6. SHALL contain [1..3] id
    • An identifier for the event. "An internal or external identifier such as a serial number on an SAE reporting form". A sequence number is required.
  7. MAY contain zero or one [0..1] targetSiteCode
    • The anatomic location that is relevant for the event. [Study Data Tabulation Model: -LOC]
  8. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Group Identifier (templateId: 2.16.840.1.113883.10.20.23.63)
  9. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Event or Finding Severity (templateId: 2.16.840.1.113883.10.20.23.61)
  10. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Pre-Specified Event (templateId: 2.16.840.1.113883.10.20.23.66)
  11. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Study Day Period (templateId: 2.16.840.1.113883.10.20.23.70)
  12. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Body System or Organ Class (templateId: 2.16.840.1.113883.10.20.23.57)
  13. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Comment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  14. SHALL contain exactly one [1..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Domain Assignment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  15. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Related Record (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  16. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Supplemental Value (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  17. MAY contain zero or one [0..1] statusCode
    • Used to indicate whether the event did occur. Use statusCode "Completed" if the event in question occurred. [Study Data Tablulation Model: -OCCUR]
  18. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Data Collection (templateId: 2.16.840.1.113883.10.20.23.59)
  19. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Non Performance Reason (templateId: 2.16.840.1.113883.10.20.23.64)
  20. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Serious Event (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  21. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Study Treatment Action Taken (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  22. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Other Treatment Action Taken (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  23. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Study Treatment Causality (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  24. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Non-Study Treatment Relationship (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  25. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Event Pattern (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  26. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Event Outcome (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  27. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Concomitant Treatment (templateId: 2.16.840.1.113883.10.20.25.xxxx)
  28. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Toxicity (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  29. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Category (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  30. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Sub- Category (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  31. SHALL satisfy: Capture the reported term - the verbatim name - using the original text property of the CE data type within the observation.value element.
  32. SHALL satisfy: Effective time information for the adverse event includes start date/time, stop date/time, and duration. Since any two of these determine the value of the third, only two of these properties of time are collected using the IVL_TS data type.

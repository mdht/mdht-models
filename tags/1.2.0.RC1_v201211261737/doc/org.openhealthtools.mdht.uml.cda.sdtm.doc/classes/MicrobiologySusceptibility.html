Microbiology Susceptibility

[Observation: templateId 2.16.840.1.113883.10.20.23.38]

The structure is used to store any findings related to the organisms found and submitted under the heading of microbiology specimen. This will usually consist of susceptibility testing results, but can also be other organism-related findings such as extent of growth of an organism.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • A coded value that identifies the microbiology susceptibility result. The verbatim text is placed within the original text property of the CD data type. [SDTM: MSTESTCD, MSTEST. MSLOINC] If the LOINC code is available it should be used as the base code within the CD type.
  4. SHALL contain [1..3] id
    • A value that is provided to ensure the uniqueness of the microbiology finding. [SDTM: MSSEQ, MSREFID, MSSPID] A single identifier, the sequence number, is required.
  5. SHOULD contain zero or more [0..*] methodCode
    • Indicates the method of the test or examination. [SDTM: MSMETHOD]
  6. MAY contain zero or one [0..1] statusCode
    • Used to indicate whether or not an exam has been performed. [SDTM: MSSTAT]
  7. SHOULD contain zero or more [0..*] value
    • The microbiology specimen result, using both original units and a standard format or set of units. [SDTM: MSORRES, MSORRESU, MSSTRESC, MSSTRESN, MSSTRESU]
  8. MAY contain zero or one [0..1] effectiveTime with data type TS
    • The date/time that the test was performed. [SDTM: MSDTC]
  9. MAY contain zero or one [0..1] entryRelationship
    • Used to record the study day associated with an activity taking place during a single day.
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Event Study Day (templateId: 2.16.840.1.113883.10.20.23.59)
  10. MAY contain zero or one [0..1] entryRelationship
    • Orients the performance of the susceptibility test with respect to a study defined timepoint.
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Timing Reference (templateId: 2.16.840.1.113883.10.20.23.70)
  11. MAY contain zero or one [0..1] entryRelationship
    • Allows capture of a group identifier for the finding.
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Group Identifier (templateId: 2.16.840.1.113883.10.20.23.61)
  12. SHOULD contain zero or one [0..1] entryRelationship

    Indicates a category the susceptibility related test or measure is assigned to.

    1. This entryRelationship SHALL contain exactly one [1..1] @typeCode="COMP"
    2. This entryRelationship SHALL contain exactly one [1..1] microbiologySusceptibilityFindingCategory with data type microbiology Susceptibility Finding Category
  13. MAY contain zero or one [0..1] entryRelationship

    Indicates a subcategory the susceptibility related test or measure is assigned to.

    1. This entryRelationship SHALL contain exactly one [1..1] @typeCode="COMP"
    2. This entryRelationship SHALL contain exactly one [1..1] microbiologySusceptibilityFindingSubCategory with data type microbiology Susceptibility Finding Sub Category
  14. MAY contain zero or one [0..1] study Test Performer
    • Used to record the organization performing the test.
    1. Contains exactly one [1..1] Study Test Performer (templateId: 2.16.840.1.113883.10.20.23.72)
  15. MAY contain zero or one [0..1] entryRelationship
    • Provides a reason for not capturing information for the finding.
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Non Performance Reason (templateId: 2.16.840.1.113883.10.20.23.62)
  16. MAY contain zero or one [0..1] entryRelationship
    • Used to indicate whether or not a result is considered a baseline value for the study.
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Baseline Indicator (templateId: 2.16.840.1.113883.10.20.23.53)
  17. MAY contain zero or one [0..1] entryRelationship
    • Used to indicate whether or not a data value has derived frrom the results of other tests or inquiries.
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Derived Data Indicator (templateId: 2.16.840.1.113883.10.20.23.57)

Serious Event

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

Information whether or not this is a serious event. [SDTM: ]. [Study Data Tabulation Model; -SER]

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="TBD" Serious Event (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
  4. SHALL contain exactly one [1..1] value with data type BL
    • A Boolean indicator that shows whether or not the event is considered to be serious. [SDTM: AESER]. [Study Data Tabulation Model: -SER]
  5. MAY contain zero or more [0..*] entryRelationship
    1. Such entryRelationships SHALL contain zero or one [0..1] @typeCode="COMP"
    2. Such entryRelationships SHALL contain exactly one [1..1] seriousEventType

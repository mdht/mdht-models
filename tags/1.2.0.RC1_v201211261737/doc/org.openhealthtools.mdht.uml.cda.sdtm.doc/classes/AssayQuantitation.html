Assay Quantitation

[Observation: templateId 2.16.840.1.113883.10.20.xxxx]

An indication of the lower limit of qauantitation for an assay.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="DEF" (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value identifies this as an asssay quantitation observation.
  4. SHALL contain zero or more [0..*] value with data type IVL_PQ
    • Captures the lower limit of assay quantitation. Value as the low property of the IVL data type. The units of measure are expected to match those designated as the standard units for the result. [SDTM: PCLLOQ], [SEND: PCLLOQ], [Study Data Tabulation Model: -LLOQ]

Exclusion Reason

[Observation: templateId 2.16.840.1.113883.10.20.23.76]

A resusable structure that indicates the reason for excluding a reported value from calculations.

  1. Contains exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="TBD" Baseline Indicator (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • A code value that indicates the observation indicates the reason for excluding a value from calculations.
  4. SHALL contain exactly one [1..1] value with data type BL
    • A coded value to indicate the reason for excluding a coded value from calculations.

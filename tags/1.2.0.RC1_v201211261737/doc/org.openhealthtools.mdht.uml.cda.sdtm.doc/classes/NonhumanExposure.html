Non-Human Exposure

[SubstanceAdministration: templateId 2.16.840.1.113883.10.20.23.21]

The template captures details of a non-human subject's exposure to protocol-specified study treatment. Study treatment may be any intervention that is prospectively defined as a test material within a study, and is typically but not always supplied to the subject.

  1. SHALL contain exactly one [1..1] @classCode="SBADM" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHOULD contain zero or one [0..1] doseQuantity
    • The amount of study treatment taken at a single administration. [SEND: EXDOSE, EXDOSU]
  4. MAY contain zero or one [0..1] effectiveTime
    • Information on when and how oftern the study treatment was taken. Possible values include start date, stop date, and duration. Note, only two of these need to be valued, and only two values are supported. [SEND: EXSTDTC, EXENDTC, EXDUR, EXDOSFRQ]
  5. SHALL contain exactly one [1..1] id
    • A sponsor supplied reference number for the study treatment. [SEND: EXSEQ]
  6. SHALL contain exactly one [1..1] routeCode
    • The route by which the treatment is administered. [SEND: EXROUTE]
  7. MAY contain zero or one [0..1] maxDoseQuantity
    • Information on the total daily dose of the medication. The structure also accommodates periods other than a day. [SDTM: EXDOSTOT] Note, this attribute can also be used if multiple total dosage amounts need to be reported.
  8. MAY contain zero or one [0..1] text
    • A text description of the dosing. [SEND: EXDOSTXT]
  9. MAY contain zero or one [0..1] approachSiteCode with data type CE
    • An indication of the location, e.g. left arm, where the substance was administered. [SEND: EXLOC]
  10. SHALL contain exactly one [1..1] consumable
    1. Contains exactly one [1..1] Consumable Material (templateId: 2.16.840.1.113883.10.20.23.58)
  11. SHOULD contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Timing Reference (templateId: 2.16.840.1.113883.10.20.23.74)
  12. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Study Day Period (templateId: 2.16.840.1.113883.10.20.23.70)
  13. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Comment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  14. SHALL contain exactly one [1..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Domain Assignment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  15. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Related Record (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  16. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Supplemental Value (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  17. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Dose Adjustment Reason (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  18. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Category (templateId: 2.16.840.1.113883.10.20.23.xxxx)

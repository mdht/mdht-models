EMS Situation Section

[Section: templateId 2.16.840.1.113883.17.3.10.1.9]

  1. SHALL conform to Derived CDA Section
  2. SHALL contain exactly one [1..1] code/@code="67666?8" EMS Situation (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  3. SHALL contain exactly one [1..1] text
  4. SHALL contain exactly one [1..1] title
  5. MAY contain zero or more [0..*] entry
    1. Such entrys SHALL contain exactly one [1..1] code/@code="10154?3" EMS complaint (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. Such entrys MAY contain zero or one [0..1] value with data type ST
    3. Such entrys MAY contain zero or one [0..1] entryRelationship
      1. This entryRelationship Contains exactly one [1..1] observation
        1. This observation SHALL contain exactly one [1..1] code/@code="67491?1" Complaint duration (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
        2. This observation SHALL contain exactly one [1..1] value with data type PQ

          Units SHALL be selected from UCUM time units (i.e., s, min, h, d, wk, mo, a)

    4. Such entrys MAY contain zero or one [0..1] entryRelationship
      1. This entryRelationship Contains at least one [1..*] observation
        1. Such observations SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
        2. Such observations SHALL contain exactly one [1..1] value with data type CD
    5. Such entrys SHOULD contain exactly one [1..1] targetSiteCode, where the @code SHALL be selected from ValueSet AnatomicLocation 2.16.840.1.113883.17.3.5.76 STATIC

      NEMSIS Trace: ESituation.07 Complaint site is required if complaint is type "Chief complaint"

    6. Such entrys SHOULD contain zero or one [0..1] entryRelationship

      The organ system is required if the complaint is of type "Chief complaint"

      1. This entryRelationship SHALL contain exactly one [1..1] observation
        1. This observation SHALL contain exactly one [1..1] code/@code="69468?7" Organ system (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
        2. This observation SHALL contain exactly one [1..1] value with data type CD, where the @code SHALL be selected from ValueSet OrganSystem 2.16.840.1.113883.17.3.5.77 STATIC

      The organ system is required if the complaint is of type "Chief complaint"

  6. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="TBD" (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type TS
  7. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="69467?9" Possible injury (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type BL
  8. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="67774?0" EMS Primary Symptom (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type CD/@code="ADD ICD 10"
  9. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="67493?7" Initial Patient Acuity (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type CD, where the @code SHALL be selected from ValueSet PatientStatus 2.16.840.1.113883.17.3.5.78 STATIC
  10. Contains exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="TBD" (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type TS
  11. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="67492?9" Provider Primary Impression (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain exactly one [1..1] value with data type CD/@code="ADD ICD10 CM"
  12. MAY contain zero or one [0..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="67776?5" Other symptoms (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain zero or more [0..*] value with data type CD/@code="ADD ICD-10 CM"
  13. SHALL contain exactly one [1..1] entry
    1. This entry SHALL contain exactly one [1..1] code/@code="69542-9" Provider Secondary Impressions (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
    2. This entry SHALL contain at least one [1..*] value with data type CD/@code="ADD ICD 10 CM"

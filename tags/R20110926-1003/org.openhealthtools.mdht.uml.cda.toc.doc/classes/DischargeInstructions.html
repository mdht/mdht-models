Discharge Instructions

[ClinicalDocument: templateId 1.2.3.4.5.101]

  1. SHALL conform to CDT General Header Constraints template (templateId: 2.16.840.1.113883.10.20.3)
  2. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Plan Of Care Section (templateId: 2.16.840.1.113883.3.88.11.83.124)
  3. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Allergies Reactions Section (templateId: 2.16.840.1.113883.3.88.11.83.102)
  4. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Problem List Section (templateId: 2.16.840.1.113883.3.88.11.83.103)
  5. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Discharge Diagnosis Section (templateId: 2.16.840.1.113883.3.88.11.83.111)
  6. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Advance Directives Section (templateId: 2.16.840.1.113883.3.88.11.83.116)
  7. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Immunizations Section (templateId: 2.16.840.1.113883.3.88.11.83.117)
  8. SHALL contain exactly one [1..1] component, such that
    1. Contains exactly one [1..1] HITSP Medical Equipment Section (templateId: 2.16.840.1.113883.3.88.11.83.128)

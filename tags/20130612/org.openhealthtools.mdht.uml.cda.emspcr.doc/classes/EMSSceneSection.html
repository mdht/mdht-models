EMS Scene Section

[Section: templateId 2.16.840.1.113883.17.3.10.1.8]

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.17.3.10.1.8"
  2. SHALL contain exactly one [1..1] code/@code="67665-0" EMS Scene Section (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10732)
  3. SHALL contain exactly one [1..1] title and SHOULD equal "EMS Scene" (CONF:10734)
  4. SHALL contain exactly one [1..1] text (CONF:10733)
  5. SHALL contain exactly one [1..1] entry (CONF:10736)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain exactly one [1..1] observation (CONF:10740)
      1. This observation SHALL contain exactly one [1..1] code (CONF:10741)/@code="67665-0" EMS reporter is first unit on scene (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10742)
      2. This observation SHALL contain exactly one [1..1] value with data type BL (CONF:10743)

        NEMSIS Trace: EScene.01

    3. This entry SHALL contain zero or one [0..1] observation
      1. This observation SHALL contain exactly one [1..1] code (CONF:10746)/@code="67481-2" Name of first unit on scene (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10745)
      2. This observation SHALL contain exactly one [1..1] value with data type ST (CONF:10747)

        NEMSIS Trace: EScene.02

  6. SHALL contain exactly one [1..1] entry (CONF:10738)
    1. This entry SHALL conform to Derived Entry template (templateId: 2.16.840.1.1133883.17.3.10.1.24)
    2. This entry SHALL contain exactly one [1..1] observation (CONF:10748)
      1. This observation SHALL contain exactly one [1..1] code (CONF:10750)/@code="67489-5" EMS Count of patients at scene (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:10749)
      2. This observation SHALL contain exactly one [1..1] value with data type INT (CONF:10751)

        NEMSIS Trace: EScene.06

    3. This entry SHALL contain zero or one [0..1] observation
      1. This observation SHALL contain exactly one [1..1] code (CONF:10753)
      2. This observation SHALL contain exactly one [1..1] value with data type BL (CONF:10754)

        NEMSIS Trace: EScene.04

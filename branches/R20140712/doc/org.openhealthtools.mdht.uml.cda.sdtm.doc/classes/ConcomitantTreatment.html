Concomitant Treatment

[Observation: templateId 2.16.840.1.113883.10.20.25.xxxx]

Information on whether another treatment was provided to the study subject as a result of an event occurrence.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="C83199" Adverse Event Concommitant Treatment (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value indicates the observation contains concomitant or additional treatment information.
  4. SHALL contain exactly one [1..1] value with data type BL
    • Indicate whether or not concomitant or additional treatment was given because of the occurrence of the event. [SDTM: AECONTRT], Study Data Tabulation Model: -CONTRT]

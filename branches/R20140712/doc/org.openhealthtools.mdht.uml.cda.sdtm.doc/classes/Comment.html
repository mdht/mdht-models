Comment

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

The template is used for submitting free-text comments. A comment may be related to a particular data item or collected seperately on a form dedicated to comments. Comments are generally not responses to specific questions; instead, comments usually consist of voluntary, free-text or unsolicited observations.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHOULD contain exactly one [1..1] code
    • Use the value "ASSERTION"
  4. MAY contain zero or one [0..1] effectiveTime
    • The date time of the comment. [SDTM: CODTC], [SEND: CODTC]
  5. SHALL contain exactly one [1..1] id
    • A sequence number for the comment. [SDTM: COSEQ], [SEND: CPOSEQ]
  6. SHALL contain exactly one [1..1] value with data type ED
    • The comment text. [SDTM: COVAL], [SEND: COVAL]
  7. SHOULD contain zero or one [0..1] participant
    • The author of the comment.
    1. Contains exactly one [1..1] Study Finding Evaluator (templateId: 2.16.840.1.113883.10.20.23.72)
  8. MAY contain zero or one [0..1] entryRelationship
    1. This entryRelationship Contains exactly one [1..1] @typeCode
    2. This entryRelationship Contains zero or one [0..1] act, where its type is CDA Act
    3. This entryRelationship SHALL contain exactly one [1..1] act
      1. This act SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
      2. This act SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
      3. This act SHALL contain exactly one [1..1] code

        Indicates the nature of the item that the comment refers to. SDTM notes: "May be the CRF page number (e.g. 650), or a module name (e.g. DEMOG), or a combination of information that identifies the reference (e.g. 650-VITALS-VISIT 2)". [SDTM: COREF], [SEND: COREF]

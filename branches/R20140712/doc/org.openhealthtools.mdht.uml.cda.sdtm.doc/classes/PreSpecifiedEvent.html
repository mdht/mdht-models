Pre-Specified Event

[Observation: templateId 2.16.840.1.113883.10.20.23.64]

An observation that indicates whther a pre-speicified event occurred or not. This structure is captured as a template to facilitate its reuse as a characteristic of different types of data within clinical trial reporting.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value indicates that the observation refers to occurrence of a pre-specified event.
  4. SHALL contain exactly one [1..1] value with data type BL
    • Indicate whether or not the the indicated activity or event was prespecified on the CRF. [SDTM: CMPRESP. SUPRESP, AEPRESP, MHPRESP, CEPRESP]

Human Clinical Subject Data Document Body

[Section: templateId 2.16.840.1.113883.10.20.23.13]

The single section which encompasses the data reported for a subject. Much of the of the content is organized by visit.

  1. SHALL contain exactly one [1..1] @classCode="DOCSECT" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain zero or one [0..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="TBD" (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  4. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Human Clinical Subject Demographics (templateId: 2.16.840.1.113883.10.20.23.14)
  5. SHOULD contain zero or more [0..*] entry
    1. Contains exactly one [1..1] Visit (templateId: 2.16.840.1.113883.10.20.23.17)
  6. SHALL contain exactly one [1..1] entry
    1. Contains exactly one [1..1] Study Arm (templateId: 2.16.840.1.113883.10.20.23.66)
  7. MAY contain zero or more [0..*] exposureAssociation
    1. Such exposureAssociations SHALL contain exactly one [1..1] @typeCode="COMP"
    2. Such exposureAssociations SHALL contain exactly one [1..1] substanceAdministration, where its type is Human Clinical Exposure
      1. Contains exactly one [1..1] Human Clinical Exposure (templateId: 2.16.840.1.113883.10.20.23.19)
  8. SHOULD contain zero or more [0..*] entry

  9. MAY contain zero or more [0..*] entry

  10. SHOULD contain zero or more [0..*] entry

  11. SHOULD contain zero or more [0..*] entry

  12. MAY contain zero or more [0..*] entry

  14. MAY contain zero or more [0..*] entry

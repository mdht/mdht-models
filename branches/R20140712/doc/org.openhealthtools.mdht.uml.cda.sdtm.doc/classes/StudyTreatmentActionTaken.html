Study Treatment Action Taken

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

A description of any changes made to the study treatment as a result of the event.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code/@code="NCI" (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
  4. SHALL contain exactly one [1..1] value with data type CE
    • Information on an action taken with the study treatment as a result of the event. [SDTM: AEACN]. [Study Data Tabulation Model: -ACN]

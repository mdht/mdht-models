Death Relationship

[Observation: templateId 2.16.840.1.113883.10.20.23.xxxx]

A description of the relationship of a particular finding to the death of a study subject.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code, where the @code SHALL be selected from (CodeSystem: 2.16.840.1.113883.3.26.1.1 NCI Thesaurus)
    • The code value indicates this is a death relationship observation.
  4. SHALL contain exactly one [1..1] value with data type CD
    • An indication of the relationship of the finding to the death of the subject. [SEND: MADTHREL, MIDTHREL, TFDTHREL], [Study Data Tabulation Model: -DTHREL]

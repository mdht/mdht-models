Microscopic Finding

[Observation: templateId 2.16.840.1.113883.10.20.23.49]

Microscopic findings for a study subject.

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood)
  3. SHALL contain exactly one [1..1] code
    • A value to indicate the particular microscopic finding observation. [SEND: MITESTCD, MITEST]
  4. SHALL contain exactly one [1..1] id
    • A sequence number for the observation. [SEND: MISEQ]
  5. SHOULD contain zero or one [0..1] value with data type CD
    • The recorded microscopic finding. [SEND: MIORRES, MISTRESC]
  6. MAY contain zero or one [0..1] statusCode
    • Used to indicate whether or not a measurement has been performed. [SEND: MISTAT]
  7. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Non Performance Reason (templateId: 2.16.840.1.113883.10.20.23.64)
  8. MAY contain zero or one [0..1] participant
    1. Contains exactly one [1..1] Study Test Organization (templateId: 2.16.840.1.113883.10.20.23.73)
  9. SHALL contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Specimen Information (templateId: 2.16.840.1.113883.10.20.23.67)
  10. MAY contain zero or one [0..1] methodCode
    • Indicates the method of the test or examination. [SEND: MIMETHOD]
  11. MAY contain zero or one [0..1] participant
    1. Contains exactly one [1..1] Study Finding Evaluator (templateId: 2.16.840.1.113883.10.20.23.72)
  12. SHOULD contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Event or Finding Severity (templateId: 2.16.840.1.113883.10.20.23.61)
  13. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Data Collection (templateId: 2.16.840.1.113883.10.20.23.59)
  14. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Group Identifier (templateId: 2.16.840.1.113883.10.20.23.63)
  15. MAY contain zero or one [0..1] participant
    1. Contains exactly one [1..1] Mass Identification (templateId: 2.16.840.1.113883.10.20.23.75)
  16. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Body System or Organ Class (templateId: 2.16.840.1.113883.10.20.23.57)
  17. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Comment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  18. SHALL contain exactly one [1..1] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Domain Assignment (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  19. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="REFR" REFR
    2. Contains exactly one [1..1] Related Record (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  20. MAY contain zero or more [0..*] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Supplemental Value (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  21. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Result Category (templateId: 2.16.840.1.113883.10.20.23.xxxx)
  22. MAY contain zero or one [0..1] entryRelationship
    1. Contains @typeCode="COMP" COMP
    2. Contains exactly one [1..1] Death Relationship (templateId: 2.16.840.1.113883.10.20.23.xxxx)

Procedure Encounter

[Encounter: templateId null]

  1. SHALL contain exactly one [1..1] @classCode="ENC" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:7771)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:7772)
  3. SHALL contain exactly one [1..1] id (CONF:7773)
  4. MAY satisfy: Set the encounter ID to the ID of an encounter in another section to signify they are the same encounter. (CONF:7774)

Code System SNOMEDCT - 2.16.840.1.113883.6.96
Description The value set contains the set of codes used to indicate the role played by the decedent with reference to a vehicle involved with a transportation injury. Note, the concept "other" is supported using the HL7 null flavor "OTH".

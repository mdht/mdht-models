Neurologic System Section

[Section: templateId 1.3.6.1.4.1.19376.1.5.3.1.1.9.35]

The neurologic system section shall contain a description of any type of neurologic exam.

  1. SHALL contain exactly one [1..1] code/@code="10202-0" NEUROLOGIC SYSTEM (CodeSystem: 2.16.840.1.113883.6.1 LOINC)
  2. MAY contain zero or one [0..1] entry, such that
    1. Contains exactly one [1..1] Problem Entry (templateId: 1.3.6.1.4.1.19376.1.5.3.1.4.5)

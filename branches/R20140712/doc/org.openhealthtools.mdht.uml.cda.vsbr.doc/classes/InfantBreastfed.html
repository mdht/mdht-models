Infant Breastfed

[Observation: templateId 2.16.840.1.113883.10.20.26.27]

Information on whether the infant was being breastfed during the period between birth and discharge from the hospital.

  1. SHALL contain exactly one [1..1] templateId ( ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.26.27"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:390)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:392)
  4. SHALL contain exactly one [1..1] code (CONF:391)/@code="73756-9" Infant is being breastfed at discharge (CodeSystem: 2.16.840.1.113883.6.1 LOINC)

    • A code value that indicates the nature of the observation - that it indicates whether the infant is being breastfed.

  5. SHALL contain exactly one [1..1] value with data type BL (CONF:393)

    • Information to identify whether the infant was being breastfed at discharge.

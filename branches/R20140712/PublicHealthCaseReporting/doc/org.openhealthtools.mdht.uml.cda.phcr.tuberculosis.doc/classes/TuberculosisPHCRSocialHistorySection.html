Tuberculosis PHCR Social History Section

[Section: templateId 2.16.840.1.113883.10.20.15.2.7]

  1. SHALL conform to PHCR Phcr Social History Section template (templateId: 2.16.840.1.113883.10.20.15.2.22)
  2. SHOULD contain zero or more [0..*] entry, such that
    1. Contains @typeCode="DRIV" DRIV (is derived from)
    2. Contains exactly one [1..1] Tuberculosis Possible Exposure Location Act (templateId: 2.16.840.1.113883.10.20.15.3.12)
  3. SHOULD contain zero or more [0..*] entry, such that
    1. Contains @typeCode="DRIV" DRIV (is derived from)
    2. Contains exactly one [1..1] Tuberculosis Socio Behavioral Boolean Risk Factor Observation (templateId: 2.16.840.1.113883.10.20.15.3.15)
  4. SHOULD contain zero or more [0..*] entry, such that
    1. Contains @typeCode="DRIV" DRIV (is derived from)
    2. Contains exactly one [1..1] Smoking Status Observation (templateId: 2.16.840.1.113883.10.20.15.3.85)
  5. SHOULD contain zero or one [0..1] entry, such that
    1. Contains @typeCode="DRIV" DRIV (is derived from)
    2. Contains exactly one [1..1] HIV Status Observation (templateId: 2.16.840.1.113883.10.20.15.3.4)
  6. SHOULD contain zero or one [0..1] entry, such that
    1. Contains @typeCode="DRIV" DRIV (is derived from)
    2. Contains exactly one [1..1] Tuberculosis Site Of TB Disease Observation (templateId: 2.16.840.1.113883.10.20.15.3.98)

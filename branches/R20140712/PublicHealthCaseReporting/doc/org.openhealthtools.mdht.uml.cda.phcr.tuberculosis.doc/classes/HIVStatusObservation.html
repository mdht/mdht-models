HIV Status Observation

[Observation: templateId 2.16.840.1.113883.10.20.15.3.4]

  1. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:861)
  2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:862)
  3. SHALL contain exactly one [1..1] code/@code="ASSERTION" (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode) (CONF:864)
  4. SHALL contain exactly one [1..1] statusCode/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:865)
  5. SHALL contain exactly one [1..1] value, which SHALL be selected from ValueSet 2.16.840.1.114222.4.11.3232 HIV Status (PHCR) STATIC 1, where its data type is CD (CONF:867)
